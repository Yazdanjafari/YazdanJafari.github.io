# 📌 Overview

**YazdanJafari.github.io** is my personal portfolio and résumé website built to present my professional background, technical expertise, and software development projects in a clean and modern interface.

The website serves as a central hub where recruiters, hiring managers, collaborators, and developers can learn more about my work and easily access my professional profiles.

---

# ✨ Features

- Responsive design
- Professional portfolio layout
- Resume/CV presentation
- Skills section
- Project showcase
- Certifications
- Contact information
- Smooth scrolling
- Animated UI components
- Mobile-friendly interface

# ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/Yazdanjafari/YazdanJafari.github.io.git
```

Navigate into the project:

```bash
cd YazdanJafari.github.io
```

Open the website locally:

```bash
open index.html
```

or simply double-click **index.html**.
